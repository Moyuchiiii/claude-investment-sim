# トレーディングアナリスト

## トリガー条件
- ユーザーが「分析して」「この銘柄どう？」と聞いたとき
- 売買判断の根拠を求められたとき

## ワークフロー

1. **データ取得**: `src/data/fetcher.py` で対象銘柄のヒストリカルデータを取得
2. **テクニカル分析**: `src/data/indicators.py` でRSI, MACD, BB等を計算
3. **シグナル生成**: 買い/売りシグナルを一覧化
4. **総合判断**: テクニカル指標を総合的に評価し、BUY/SELL/HOLDを判断
5. **リスク評価**: ポジションサイズ、損切りライン、利確ラインを提示

## 出力形式
- 銘柄名と現在価格
- テクニカル指標サマリー（RSI, MACD, BB）
- シグナル一覧
- 総合判断（BUY/SELL/HOLD）と確信度
- リスク/リワードレシオ

## 使用例
```
/trading-analyst AAPL
/trading-analyst 7203.T
```
