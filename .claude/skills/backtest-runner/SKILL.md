# バックテストランナー

## トリガー条件
- ユーザーが「バックテストして」と依頼したとき
- 新しい戦略を検証したいとき

## ワークフロー

1. **パラメータ確認**: 銘柄、期間、戦略パラメータを確認
2. **バックテスト実行**: `python scripts/backtest.py` を実行
3. **結果分析**: リターン、勝率、シャープレシオ、最大DDを評価
4. **改善提案**: パラメータ調整や戦略変更の提案

## 使用例
```
/backtest-runner           # 全銘柄で1年分
/backtest-runner AAPL 2y   # AAPL で2年分
```
